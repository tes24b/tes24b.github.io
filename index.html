Project 01
8Ball Sports — Interactive Sports Dashboard
Web Design UI/UX Frontend Data Display
Visual representation
The 8Ball Sports dashboard is a fully functional web application built to display real-time sports data. The platform covers live game scores, individual player statistics, team records, game history, and predictive analytics, all wrapped in a sleek dark-themed interface designed for sports fans who want depth without clutter.
Project goals & scope
The goal was to create a prototype sports platform that felt as polished and functional as a production product. From the Home page down to Predictions and History, every section needed to feel cohesive and purposeful. The scope covered UI design, layout architecture, and how data was presented across six distinct pages, each serving a different type of sports information.
My contribution
I took ownership of the visual design and layout across the entire application. My focus was making sure every page felt like it belonged to the same product with consistent spacing, typography, color usage, and component structure throughout. The challenge was keeping the interface complex enough to be useful while making sure it never felt overwhelming to the user. I designed the system from the ground up and applied it across all six pages.
Project 02
FSU CCI Website — UX Research & Analysis
UX Research Data Analysis User Interviews Usability Study
Visual representation
This journey map captures the full arc of our UX research process for the FSU CCI website. It traces the path from initial site review through focus group design, live user interviews, data analysis, and final recommendations, showing how raw user feedback was transformed into actionable insights.
Project goals & scope
The goal of this project was to conduct a thorough UX audit of the FSU College of Communication and Information website and identify what was not working for students. The team used a range of research methods including online surveys through Qualtrics, one-on-one user interviews, usability testing sessions, and structured focus groups. Through this process the team surfaced consistent issues around navigation, visual design, accessibility, and the relevance of content being served to students versus faculty and alumni.
My contribution
I served as the lead interviewer for the project, personally designing the interview questions and running the sessions with students and stakeholders. Beyond the interviews I led the data analysis phase, working through open, axial, and selective coding to pull meaningful themes out of the qualitative responses we collected. I then translated those findings into the recommendations and insights that shaped the journey map and the final presentation delivered by our team.
